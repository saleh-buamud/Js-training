## Features Added

**Name Input Preview and List Append**

- **Preview on Click:** Display the input value in the preview box when the container is clicked.  
- **Add to List:** Add the input value as a new `<li>` item to the list when the "Add" button is clicked.  
- **Clear Input & Preview:** Automatically clear the input field and preview after adding the item.  
- **Improved Naming:** Variable names have been updated for better clarity and readability.
